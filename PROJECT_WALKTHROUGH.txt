                MERN Stack Task Management Application

PROJECT OVERVIEW
================================================================================
TaskFlow is a full-stack task management and team collaboration platform built
using the MERN stack (MongoDB, Express.js, React, Node.js). The application
enables users to manage personal tasks, create teams, collaborate on team
projects using Kanban boards, track activity, receive notifications, and more.

This document demonstrates how all final project requirements have been met,
organized by the grading rubric categories totaling 180 points.

--------------------------------
- Framework: React 18.2.0
- Build Tool: Vite 5.0.8
